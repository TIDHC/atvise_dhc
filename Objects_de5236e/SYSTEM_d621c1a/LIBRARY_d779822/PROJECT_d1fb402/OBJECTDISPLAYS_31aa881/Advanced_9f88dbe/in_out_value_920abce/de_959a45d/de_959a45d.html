Beschreibung

Diese erweiterte in_out_value Komponente ist eine modifizierte Version und besitzt 3 zusätzliche Einstellungsparameter.
Es kann zum Eingeben von Werten über eine physische Tastatur oder die Popup-SVG-Tastatur verwendet werden.
Ebenfalls kann auch eine Textbeschränkung und die Ausrichtung des angezeigten Textes festgelegt werden.
Diese Komponente verwendet die folgenden zusätzlichen Konfigurationsparameter neben den Standardparametern der Komponente in_out_value:

Parameter

Diese erweiterte in_out_value - Komponente hat folgende Zusatzparameter:

  • text align: Legt die Ausrichtung von Text links oder rechts fest. (ENUM)
  • maximum textlength: Gibt die maximal mögliche Länge von Textzeichen an. Wenn das Maximum erreicht ist, wird eine Fehlermeldung im SVGKeyboard angezeigt. (INT/NUMBER)
  • replace text on typing: Steuert, ob beim Eingeben eines Zeichens der gesamte Text im SVGKeyboard oder im Direkteingabemodus ersetzt wird oder nur die neuen eingegebenen Zeichen am Ende des Textes hinzugefügt werden. (BOOL)
