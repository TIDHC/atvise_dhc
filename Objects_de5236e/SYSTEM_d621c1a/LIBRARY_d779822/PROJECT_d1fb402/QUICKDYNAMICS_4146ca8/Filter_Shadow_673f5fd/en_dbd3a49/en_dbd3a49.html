Description - Apply SVG filter for shadow drop generation

This Quick Dynamic generates shadow behind the SVG element to which it is applied.

Parameters:

  • Shadow offset X: The x offset for the shadow (in pixels, relatively to the source).
  • Shadow offset Y: The y offset for the shadow (in pixels, relatively to the source).
  • Blur Level of Shadow Edges: Gaussian blur index ([0]: No blur; [1,2,3...]: Blur level)
  • Input type:
    • SourceAlpha: Takes alpha-channel as the source (grayscale, preserves transparency)
    • SourceGraphics: Takes RGB component as the source (colorful if applied to simple SVG element)
  • Shadow Intensity Level:([0]: No Shadow/Fully transparent - [100]: Dark Shadow)
