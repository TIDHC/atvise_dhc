Description

This in_out_value component is a modified version and includes new parameters.
It can be used for typing in values by physical keyboard or a popup SVG Keyboard.
Also a text limitation can be set and the alignment of the displayed text.
This component uses the following extra configuration parameters beside the default parameters of standard in_out_value component:

Parameters

This control has the following extra parameters:

  • text align: Sets the alignment of text, left or right. (ENUM)
  • maximum textlength: Gives the maximum possible length of text characters. If the maximum is reached an error message will be displayed in the SVGKeyboard. (Expects INT/NUMBER)
  • replace text on typing: Controls if on typing a character the whole text in the SVGKeyboard or direct input mode will be replaced or just adds the new typed characters in the end of the text. (Expects BOOL)
